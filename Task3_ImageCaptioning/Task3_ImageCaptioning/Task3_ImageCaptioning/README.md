# Image Captioning AI

This project is part of the CodSoft Artificial Intelligence Internship.

## Description
This is an AI-based application that generates captions for images using deep learning models.

## Technologies Used
- Python
- Streamlit
- Transformers
- Torch
- Pillow

## Features
- Upload image
- Generate caption automatically
- User-friendly interface

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py
