# CODSOFT Artificial Intelligence Internship

This repository contains the tasks completed during my Artificial Intelligence Internship at CodSoft.
The purpose of this internship is to learn and apply AI concepts through practical projects.

---

## Task 1: Rule-Based Chatbot

Description:
A simple chatbot built using Python that responds to user inputs based on predefined rules.
It uses conditional statements to identify user queries and provide appropriate responses.

Technologies Used:

* Python
* If-Else Statements
* Basic Natural Language Processing Concepts

Features:

* Responds to greetings
* Answers basic questions
* Can perform simple calculations
* Displays date and time
* Basic interactive conversation

---

## Task 2: Tic-Tac-Toe AI

Description:
A Tic-Tac-Toe game where a human player can play against an AI opponent.
The AI makes intelligent moves based on game logic.

Technologies Used:

* HTML
* CSS
* JavaScript

Features:

* Human vs AI gameplay
* Interactive user interface
* Game result detection (Win, Lose, Draw)
* Restart game functionality
